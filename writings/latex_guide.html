Equations

The following are some useful tools when changing the formatting of equations.

Tags - https://ctan.org/pkg/mathtools?lang=en

Tags can be used to change what numbers your equations. The default is standard arabic numerals, but with tags you can change these to any other number of letter system. The syntax for this is, \newtagform{name}[\renewcommand{\theequation}{\style{equation}}]() Where name is the custom name you want to use when calling this tag in the body of your document, and style is the numbering style you want to use.
